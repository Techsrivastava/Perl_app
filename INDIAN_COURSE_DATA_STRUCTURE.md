# Indian Course Data Structure - Complete Documentation

## Overview
This document details the comprehensive India-specific course data structure implemented in the University Management System app. The system now includes **70+ fields** covering all aspects of Indian higher education.

### Basic Course Information
1. **id** - Unique course identifier
2. **universityId** - Associated university ID
3. **name** - Full course name
4. **code** - Course code (e.g., CSE-BTECH-2024)
5. **department** - Department/Stream name
6. **degreeType** - Degree type (B.Tech, MBBS, MBA, etc.)
7. **duration** - Course duration
8. **modeOfStudy** - Full-time/Part-time/Distance/Online
9. **level** - Undergraduate/Postgraduate/Doctoral
10. **fees** - Annual fees in INR
11. **totalSeats** - Total available seats
12. **availableSeats** - Currently available seats

## üìç India-Specific Data Points

### Entrance Exams Covered
- **Engineering**: JEE Main, JEE Advanced, State CETs
- **Medical**: NEET-UG, NEET-PG, AIIMS
- **Management**: CAT, MAT, XAT, CMAT, SNAP, NMAT
- **Law**: CLAT, AILET, LSAT
- **Design**: NIFT, NID, UCEED
- **Architecture**: NATA, JEE Main Paper 2
- **Pharmacy**: GPAT, State entrance tests
- **Agriculture**: ICAR AIEEA

### Accreditation Bodies
- **NAAC** - National Assessment and Accreditation Council
- **NBA** - National Board of Accreditation
- **UGC** - University Grants Commission
- **AICTE** - All India Council for Technical Education
- **MCI/NMC** - Medical Council of India/National Medical Commission
- **DCI** - Dental Council of India
- **BCI** - Bar Council of India
- **PCI** - Pharmacy Council of India
- **INC** - Indian Nursing Council
- **COA** - Council of Architecture
- **NCTE** - National Council for Teacher Education

### University Types
- **Central Universities** - Funded by central government
- **State Universities** - Funded by state government
- **Deemed Universities** - Deemed-to-be-universities
- **Private Universities** - Self-financed
- **Autonomous Colleges** - Affiliated but autonomous

### Reservation Categories
- **SC** - Scheduled Caste (15%)
- **ST** - Scheduled Tribe (7.5%)
- **OBC** - Other Backward Classes (27%)
- **EWS** - Economically Weaker Section (10%)
- **PWD** - Persons with Disabilities (5%)

### NAAC Grades
- **A++** - CGPA 3.51-4.00 (Excellent)
- **A+** - CGPA 3.26-3.50 (Very Good)
- **A** - CGPA 3.01-3.25 (Good)
- **B++** - CGPA 2.76-3.00
- **B+** - CGPA 2.51-2.75
- **B** - CGPA 2.01-2.50
- **C** - CGPA 1.51-2.00
